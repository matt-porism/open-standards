Creating and maintaining open standards

Guidance on processes and techniques for developing open standards for data. Written and maintained by Porism from content developed for and owned by the Open Data Institute with contributions from many others.

  1. What is a standard?

    The formal definition of a standard, and specifically a data standards as covered by this website

  2. Processes for open standards

    Guidance on when and why we might need standards and, how we go about creating them

  3. Techniques

    The technicalities of creating data standards, their metadata and properties

  4. Tools

    Resources that can be used to help with data modelling, expressing a standard in different formats, and validation of data against a standards
